OpenAI has expanded its Responses API with new capabilities including shell command execution, agent skills framework, and server-side compaction features. These additions enable developers to build more sophisticated AI agents that can directly interact with system environments. The shell tool integration allows AI to execute infrastructure operations safely. Server-side compaction reduces payload sizes and improves API efficiency for large-scale deployments.
