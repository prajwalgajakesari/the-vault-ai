Anthropic inadvertently exposed 512,000 lines of proprietary Claude Code source code through an npm package publication error. The leak included internal tooling, development utilities, and architectural details of the Claude Code system. Anthropic rapidly responded to secure and remove the exposed materials from public repositories. The incident highlights security challenges in managing proprietary code across open-source distribution channels.
