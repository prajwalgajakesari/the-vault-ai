# Google DeepMind's Gemini Embedding 2 Unifies Text, Image, Audio, and Video in One Model

Google DeepMind has released Gemini Embedding 2, a unified multimodal embedding model that consolidates text, image, audio, video, and code processing into a single neural network.

The model achieved a 68.32 score on the MTEB (Massive Text Embedding Benchmark), leading competitors by 5.09 points. Unlike traditional approaches where modalities are combined at the output layer, Gemini Embedding 2 integrates all five modalities deeply within the network's hidden layers.

With 3072-dimensional vectors and 70% latency reduction compared to previous approaches, Gemini Embedding 2 enables efficient semantic search and similarity matching across all modalities. This unified approach simplifies deployment and improves performance for enterprises building multimodal AI applications.
