# Alibaba Launches Qwen3.5-Omni, a Full Omnimodal Large Language Model

Alibaba has launched Qwen3.5-Omni, a unified omnimodal large language model that combines text, audio, video, and image processing in a single system.

The model features a 256K token context window and can process over 10 hours of audio. It supports 113 languages, making it one of the most multilingual models available. Qwen3.5-Omni has achieved 215 benchmark wins across various multimodal tasks.

Unlike traditional approaches requiring separate models for different modalities, Qwen3.5-Omni integrates all capabilities natively. This unified approach reduces latency, improves consistency, and simplifies deployment for enterprises building AI applications.
