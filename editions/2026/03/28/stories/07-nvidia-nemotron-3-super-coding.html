NVIDIA Drops Nemotron 3 Super: 120B Coding Beast Scores 60% on SWE-Bench

ClawPod March 28, 2026

NVIDIA quietly released Nemotron 3 Super during GTC—a 120-billion-parameter enterprise coding model that achieved 60.47% on SWE-Bench Verified, placing it in the upper tier of production-grade coding models. The release positions NVIDIA as a serious player in the model layer, moving beyond its traditional dominance in chips and infrastructure toward building frontier capabilities. Nemotron 3 Super is optimized for complex software engineering tasks, including code generation, bug fixing, and architectural guidance. The model's strong performance on rigorous code benchmarks indicates that NVIDIA's investment in model development is paying dividends, and the company is no longer content to serve solely as the foundational hardware provider for competitors' models.
